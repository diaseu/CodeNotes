HTML

  • bookmark_borderHTML Basics

    HTML (Hyper Text Markup Language) is the standard markup language for websites. HTML elements are building blocks of HTML pages, and are represented by < > tags.

    These are some examples of basic HTML elements.
    • The <!DOCTYPE html> declaration defines that this document is an HTML5 document.
    • The <html> element is the root element of an HTML page.
    • The <head> element contains meta information about the HTML page.
    • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
    • The <body> element defines the document's body, and is a container for all the visible contents, such as headings,paragraphs, images, hyperlinks, tables, lists, etc.
    • The <h1> element defines a large heading
    • The <p> element defines a paragraph

    All websites should have at least the following elements: !DOCTYPE html declaration, html, head, title, body.
    In VS Code, any HTML file types can have the basics populated by typing in ! and pressing enter.

  • placeSemantic Elements
    A semantic element clearly declares its purpose to the developer, reader and browser. Semantic web allows data to be shared and reused across applications, enterprises and communities. There is no restricting on nesting semantic elements, so article can be nested inside section and vice versa.

    Examples of semantic elements include:

    • <article> - independent, self-contained content, like a blog post or newspaper article
    • <aside> - content aside from page content
    • <details> - additional details that user can show or hide
    • <figcaption> - caption for figure
    • <figure> - self-contained content like photos, diagrams, code listings, etc
    • <footer> - footer typically contains authorship, copyright, contact, sitemap, back to top, etc. You can have multiple footers
    • <header> - container for introductory content or set of navigational links. Cannot be placed in footer, address or header
    • <main> - main content
    • <mark> - marked/highlighted text
    • <nav> - navigation links
    • <section> - thematic grouping of contenting, typically with a heading
    • <summary> - visible heading for details element
    • <time> - time
    • <table>
    • <form>

    Non-semantic elements, to compare, are things like:

    • <div>
    • <span>

CSS

  • bookmark_borderCSS Basics
    • Syntax of CSS:
      selector {property: value; property: value;}
      h1 {
      color: blue;
      font-size: 24px;
      }
    • Order of css matters (within stylesheet) - latter CSS overwrites earlier CSS
    • Similarly, order of css stylesheets matters too - latter CSS stylesheets overwrites earlier CSS stylesheets
    • Browsers often come with some default values for many css elements. Use a reset "primer" to wipe out those default values. (Link to reset.css)
  • pending_actionsTypes of CSS Selectors
    1. CSS Element Selector
      p {
         text-align: center;
         color: red;
      }
    2. CSS id Selector
      #first {
         text-align: center;
         color: red;
      }
    3. CSS class Selector
      .center {
         text-align: center;
      }
    4. CSS Universal Selector
      * {
         text-align: center;
      }
    5. CSS Grouping Selector
      h1, h2, p {
         text-align: center;
      }
  • sync_altApplying CSS to HTML

    To apply CSS to HTML, the first important aspect is to include in <head>:

    1. link to the stylesheet (preferred)
      <link rel="stylesheet" href="./assets/css/style.css">
      or
    2. <style> tag with css directly
      <head>
         <style>
           p {
             color: red;
           }
         </style>
      </head>



    The next important step is to declare classes and IDs:

    1. Classes:
      Classes applied in HTML (with stylesheet link in head):
      <p class="red">Hello World</p>
      Classes declared in CSS (style.css):
      .red {
        color: red;
      }
      Result:

      Hello World

    2. IDs:
      IDs applied in HTML (with stylesheet link in head):
      <p id="first">Hello World</p>
      IDs declared in CSS (style.css):
      #first {
        color: blue;
      }
      Result:

      Hello World
